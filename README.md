# plan-nest-fe
포트폴리오 - 여행 &amp; 부동산 등 계획이 모이는 통합 플랫폼 front-end


## 프로젝트 스택
 - 상태 관리: Zustand
 - 스타일링: Tailwind CSS
 - API 요청: React Query
 - 라우팅: React Router / Next.js
 - 폼 관리: React Hook Form
 - UI 컴포넌트: ShadCN/UI
 - 아이콘: Lucide Icons
 - 차트: Recharts
